# Request Page — La Cruz Oficial
A modern request page built for a church community, designed to allow users to submit prayer requests in a simple a■ Live Demo: https://request-page-lacruz-oficial.vercel.app/
---
## Features
- Request Submission — Users can send prayer requests through a simple form
- Real-Time Updates — Requests are stored and displayed dynamically
- Clean UI — Minimalist and accessible design focused on clarity
- Fast Performance — Optimized with modern frontend tooling
- Responsive Design — Works across desktop and mobile devices
---
## Tech Stack
Framework: Next.js
Language: TypeScript
Styling: Tailwind CSS
Backend: Supabase
Database: PostgreSQL (via Supabase)
Deploy: Vercel
---
## Architecture Highlights
- Supabase Integration — Backend-as-a-service for real-time data
- Serverless Approach — No traditional backend server
- Component-Based UI — Modular and reusable components
- Responsive-first Design — Mobile-first approach
---
## Getting Started
### Prerequisites
- Node.js 18+
- Supabase account
### Installation
git clone https://github.com/LucasEzequielRamos/request-page-lacruz-oficial.git
cd request-page-lacruz-oficial
npm install
### Environment Variables
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
### Run the App
npm run dev
